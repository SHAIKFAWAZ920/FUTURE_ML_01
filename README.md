# FUTURE_ML_01
Sales & Demand Forecasting using Machine Learning

## 📌 Objective
To predict future sales based on historical retail data.

## 📊 Dataset
Superstore Sales Dataset

## ⚙️ Steps Performed
- Data Cleaning
- Time-based Feature Engineering
- Random Forest Regression Model
- Model Evaluation (MAE & RMSE)
- 3-Month Future Forecast

## 📈 Results
The model predicts future monthly sales trends.
This helps businesses plan inventory, staffing, and marketing strategies.

## 💼 Business Value
Businesses can:
- Increase stock before high demand
- Reduce inventory in slow months
- Plan marketing campaigns strategically
